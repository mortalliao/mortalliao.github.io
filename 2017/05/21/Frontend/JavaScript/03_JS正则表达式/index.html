JS正则表达式

创建正则表达式对象的方式：

  1. 方式一（推荐）：/pattern/[flags]
  • pattern：正则表达式的规则；

  • 边界：

    • ^（开始）
    • $ (结束)

  • flag：模式

    • i 代表忽略大小写；
    • g 全文匹配
  1. new RegExp(“pattern”,[“flags”])

正则表达式的方法：

test(); // 匹配正则表达式的规则，如果匹配到就返回true，否则返回false；
exec(); // 查找字符串中符合正则表达式的内容；
